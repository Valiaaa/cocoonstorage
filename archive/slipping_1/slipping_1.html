This website is designed to present an expanding collection of poems that I wrote. One of the main aims of this project is: how can modern designers archive literary works in a new and sensational way? To explore this topic, I ended up using 5 different indexes to present each poem: color, word count, photo, date, and title: from unfamiliar ones to classical ones. The random show-up by every refresh not only provides a fresh experience but also reinforces my concept.
The adaptive visual grid system is something that I also focus on to make the whole UX clear and comfortable.
